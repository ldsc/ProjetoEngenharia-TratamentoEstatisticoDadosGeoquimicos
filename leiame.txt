

Organização do diretório:
====================
O presente diretório tem 3 sub-diretórios:

imagens: 
	Neste diretório você deve colocar todas as imagens a serem utilizadas e referenciadas em seu trabalho.
   
listagens:  
	Neste diretório você deve colocar todas as listagens de código,
	arquivos .h .cpp

	listagens/html:
	Após documentar o código usando formato javadoc(/// no lugar de  //), 
	você irá usar o programa doxygen para gerar automaticamente a documentação docódigo. 
	Comandos do doxygen:	
	- Abra um terminal e vá para o diretório com o progama a ser documentado usando cd.
	- Execute o comando (doxygen -y) para gerar o arquivo de configuraćão do doxygen
	- Execute o comando (doxygen) para gerar documentaćão  do código.
	O doxygen irá criar um subdiretório chamado html, nele você encontrará um arquivo index.html 
	que deve ser aberto usando um navegador (como o Chrome ou Firefox).
	- Abra o arquivo listagens/html/index.html e  veja como ficou a documentaćão, navegue pelos arquivos.
	- Note que você pode mudar os parâmetros do arquivo DOXYFILE; usando umm editor de texto abra o arquivo DXYFILE, 
	leia cada parágrafo e a seguir ative/desative determinada općão; a seguir rode novamente o doxygen, 
	e verifique como ficou a documentaćão (é bem simples). 
	Quando mais općões SIM, mais completa fica a documentaćão (e demora mais para gerar).

lyx:  
	Neste diretório você vai encontrar arquivos do editor de texto científico LyX.
